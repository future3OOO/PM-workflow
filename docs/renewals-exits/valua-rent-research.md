**Version:** V2.3  
**Last updated:** 2026-03-31

---

## Steps
1) Build 4–8 comparable properties (same area, similar bed/bath/condition)  
2) Exclude outliers  
3) Produce recommended range and suggested figure  
4) Send owner summary (range + figure + 2–3 bullet rationale)  
5) Save evidence export/screenshot into Property Tree docs (03 Compliance or a Rent Reviews subfolder)
