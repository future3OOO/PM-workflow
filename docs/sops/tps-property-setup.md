# SOP — TPS: Property Setup + Trade Me Listing ID + BookMe Routing (Version 2)

**Version:** V2 (Full)  
**Last updated:** 2026-02-24

---

## Purpose / Outcome
TPS property is configured so you can run BookMe viewings, receive applications, run checks, generate agreements, and execute renewals/extensions.

## Trigger
New property onboarded OR ready to lease.

## Steps
1) Create/confirm the property in TPS (address, basic property data)  
2) Complete TPS compliance section fields required for agreement generation  
3) Publish the Trade Me listing and capture the listing number  
4) Enter Trade Me listing ID into TPS property details to route enquiries into BookMe  
5) Confirm enquiries are appearing in BookMe dashboard

## Quality checks
- TPS property data accurate
- Compliance fields complete enough for agreement
- BookMe routing confirmed (test enquiry if needed)
