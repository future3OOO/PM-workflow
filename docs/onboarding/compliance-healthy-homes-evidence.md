# Healthy Homes Evidence (TPS)

**Version:** V2.0  
**Last updated:** 2026-02-24

---

## Purpose / Outcome
Healthy Homes (and related compliance evidence) is stored correctly in Property Tree and reflected accurately in TPS compliance fields so agreements/renewals are correct.

## Steps
1) Gather existing evidence (reports, certificates, invoices, photos, product specs)
2) Store evidence in Property Tree docs → 03 Compliance (use consistent naming)
3) Enter/update TPS compliance section fields using evidence
4) If gaps exist, create remediation tasks in Tapi with owner approval plan
5) After remediation, file completion evidence in Property Tree and update TPS fields

## Quality checks

!!! warning "Evidence integrity"
    - Evidence must exist in Property Tree for every compliance claim
    - TPS compliance fields must match the evidence — no "wishful" entries
