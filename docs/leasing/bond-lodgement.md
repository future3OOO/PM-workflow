# Bond Lodgement

**Version:** V2.3  
**Last updated:** 2026-03-31

---

## Purpose / Outcome
Bond is collected into trust first, then lodged with Tenancy Services via TPS workflow, with confirmation filed into Property Tree.

## Steps
1) Receive bond into trust account and record internally
2) Initiate bond lodgement via TPS (Tenancy Services integration)
3) Pay Tenancy Services using the provided reference number (from trust funds)
4) Track to confirmation (email/letter)
5) Upload confirmation into Property Tree docs and update bond status

## Timeframe

!!! warning "Statutory deadline"
    If tenant pays bond to landlord/agent, it **must be lodged within 23 working days** of receiving it. Track this deadline actively.
