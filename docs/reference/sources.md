  https://www.tenancy.govt.nz/rent-bond-and-bills/rent/increasing-rent/

  https://www.tenancy.govt.nz/rent-bond-and-bills/bond/lodging-a-bond/about-lodging-a-bond/
- Bond timeframe explanation (also stated in “preventing problems with bonds”).  
  https://www.tenancy.govt.nz/rent-bond-and-bills/bond/preventing-problems-with-bonds/
