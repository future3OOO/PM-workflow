# Weekly Operations Checklist

**Version:** V2.3  
**Last updated:** 2026-03-31

---

## 1) Renewals pipeline (next 120 days)

See: [Renewals & Rent Reviews](../renewals-exits/renewals-rent-reviews.md)
- [ ] List fixed terms ending
- [ ] Landlords contacted and follow-ups scheduled
- [ ] Tenants notified once landlord direction received
- [ ] Escalate non-responses (2 emails + 1 call + deadline email)

## 2) Inspections (next 30 days)

See: [Inspection Lifecycle](../inspections/inspection-lifecycle.md) | [Property Tree Scheduling](../inspections/pt-scheduling.md)
- [ ] Scheduled in Property Tree and synced
- [ ] Confirmation status checked and chased
- [ ] Overdue reports cleared

## 3) Maintenance hygiene

See: [Maintenance Lifecycle](../maintenance/maintenance-lifecycle.md) | [Tapi Intake & Triage SOP](../maintenance/tapi-intake.md)
- [ ] Open jobs sorted by age and priority
- [ ] Approvals exist where required
- [ ] Stalled jobs chased
- [ ] Completed jobs closed (evidence attached)
- [ ] Review stale open jobs (>30 days with no activity) — close or follow up
- [ ] Ensure all completed DIY jobs have been closed in Tapi
- [ ] Check for recommended maintenance items from inspections with no owner response >7 days

## 4) Invoices + payments

See: [Tapi Invoices SOP](../maintenance/tapi-invoices.md)
- [ ] All invoices coded correctly (owner/tenant/DIY)
- [ ] Tenant charges have evidence and comms trail
- [ ] Sync to Property Tree confirmed ahead of Monday run
- [ ] Review unmatched/unapproved invoices in Tapi
- [ ] Verify Property Tree sync for all approved invoices from the week

## 5) Inbox and records

See: [Email Management](email-management.md)
- [ ] Shared inbox WAITING reviewed and chased
- [ ] Spot check: 5 properties have latest key docs filed
