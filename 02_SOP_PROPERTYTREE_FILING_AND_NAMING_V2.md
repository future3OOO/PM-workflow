# SOP — Property Tree: Filing + Naming (Version 2)

**Version:** V2 (Full)  
**Last updated:** 2026-02-24

---

## Purpose
Make Property Tree the complete, searchable property file.

## Folder structure (standard)
- 01 Management & Owner Docs
- 02 Tenancy Agreements
- 03 Compliance
- 04 Inspections
- 05 Maintenance
- 06 Notices & Key Correspondence
- 07 Insurance & Certificates

## Naming conventions
Use: YYYY-MM-DD — <Doc Type> — <Short Description>

Examples:
- 2026-02-24 — Tenancy Agreement — Signed — Smith
- 2026-02-24 — Bond — Lodged Confirmation
- 2026-02-24 — Inspection — Routine — Report
- 2026-02-24 — Notice — Rent Increase — Effective 2026-05-01
- 2026-02-24 — Maintenance — Invoice — Plumber — WO12345

## Email filing
If a decision was made by email:
- export PDF
- file under 06 Notices & Key Correspondence
- add short note in property record referencing it
