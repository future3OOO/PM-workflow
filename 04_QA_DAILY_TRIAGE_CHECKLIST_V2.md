# QA — Daily Triage Checklist (Version 2)

**Version:** V2 (Full)  
**Last updated:** 2026-02-24

---

## 1) Maintenance (Tapi)
- [ ] New requests acknowledged (tenant has a next-update time)
- [ ] Urgent items issued immediately (and owner notified at same time)
- [ ] Owner approvals pending sent or chased
- [ ] Contractor follow-ups for stalled urgent jobs

## 2) Leasing (TPS/BookMe)
- [ ] Viewings have availability
- [ ] BookMe bookings reviewed (changes communicated)
- [ ] New applications triaged and next steps sent

## 3) Inspections
- [ ] Inspections within 10 days have been communicated
- [ ] Unconfirmed inspections chased (no confirmation = no attendance)

## 4) Inbox hygiene (VA)
- [ ] Clear URGENT then ACTION labels
- [ ] Chase WAITING items
- [ ] File key decisions to Property Tree
