# SOP — TPS Applications: Processing + Checks + Approval (Version 2)

**Version:** V2 (Full)  
**Last updated:** 2026-02-24

---

## Purpose / Outcome
Applications are processed consistently with a documented decision trail and appropriate use of checks.

## Rules
- Triage all applications for completeness.
- Run background/credit checks on preferred applicant(s) only.
- Capture owner decision in writing.
- Approve in TPS to trigger acceptance notifications and agreement workflow.

## Steps
1) Triage for completeness and shortlist  
2) Run TPS checks on preferred applicant(s)  
3) Provide owner recommendation summary  
4) Approve in TPS  
5) File decision summary (as permitted) to Property Tree
