# QA — Daily Triage Checklist (Version 2)

**Version:** V2.1 (Updated from video walkthrough)  
**Last updated:** 2026-03-07

---

## 1) Maintenance (Tapi)
- [ ] New requests acknowledged (tenant has a next-update time)
- [ ] Urgent items issued immediately (and owner notified at same time)
- [ ] Owner approvals pending sent or chased
- [ ] Contractor follow-ups for stalled urgent jobs

## 2) Inspection follow-up (Inspection Express → Tapi)
- [ ] Check Inspection Express for newly finalised inspections
- [ ] Review each finalised report PDF for maintenance items
- [ ] Confirm all maintenance items have synced into Tapi
- [ ] Clean up job descriptions and merge related jobs where applicable
- [ ] Check PM's email inbox for any existing owner responses to inspection reports
- [ ] Send owner approval in Tapi for items not yet responded to
- [ ] Send work order directly for items already approved by owner
- [ ] Follow up on Tapi "Awaiting approval" jobs older than 3 business days
- [ ] Flag any inspections still "In Progress" — remind PM to finalise

## 3) Leasing (TPS/BookMe)
- [ ] Viewings have availability
- [ ] BookMe bookings reviewed (changes communicated)
- [ ] New applications triaged and next steps sent

## 4) Inspections (upcoming)
- [ ] Inspections within 10 days have been communicated
- [ ] Unconfirmed inspections chased (no confirmation = no attendance)

## 5) Inbox hygiene (VA)
- [ ] Clear URGENT then ACTION labels
- [ ] Chase WAITING items
- [ ] File key decisions to Property Tree
