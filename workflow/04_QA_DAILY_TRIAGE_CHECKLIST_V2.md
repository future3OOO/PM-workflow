# QA — Daily Triage Checklist (Version 2)

**Version:** V2.2 (Updated from video walkthrough)  
**Last updated:** 2026-03-31

---

## 1) Maintenance (Tapi)
- [ ] New requests acknowledged (tenant has a next-update time)
- [ ] Urgent items issued immediately (and owner notified at same time)
- [ ] Owner approvals pending sent or chased
- [ ] Contractor follow-ups for stalled urgent jobs

## 2) Inspection follow-up (Inspection Express → Tapi)
- [ ] Check Inspection Express for newly finalised inspections
- [ ] Review each finalised report PDF for maintenance items
- [ ] Confirm all maintenance items have synced into Tapi
- [ ] Clean up job descriptions and merge related jobs where applicable
- [ ] Check PM's email inbox for any existing owner responses to inspection reports
- [ ] Send owner approval in Tapi for items not yet responded to
- [ ] Send work order directly for items already approved by owner
- [ ] Follow up on Tapi "Awaiting approval" jobs older than 3 business days
- [ ] Flag any inspections still "In Progress" — remind PM to finalise

## 3) Leasing (TPS/BookMe)
- [ ] Viewings have availability
- [ ] BookMe bookings reviewed (changes communicated)
- [ ] New applications triaged and next steps sent

## 4) Inspections (upcoming)
- [ ] Inspections within 10 days have been communicated
- [ ] Unconfirmed inspections chased (no confirmation = no attendance)

## 5) Inbox hygiene (VA)
- [ ] Clear URGENT then ACTION labels
- [ ] Chase WAITING items
- [ ] File key decisions to Property Tree

## 6) Tapi Maintenance Dashboard — Daily Check

| # | Check | Detail |
|---|-------|--------|
| 1 | Open Tapi Jobs dashboard | Review status pipeline counts: Open jobs, Awaiting quotes, Awaiting approval, Scheduling, Awaiting invoice |
| 2 | Check new Concierge requests | Look for email notifications with Concierge/Maintenance labels. Accept in Tapi, clean up, triage |
| 3 | Review "Open jobs" bucket | Any new jobs needing triage? Leave notes for PM if unsure about action |
| 4 | Review "Awaiting approval" bucket | Any approvals >3 business days old? Send follow-up via Tapi thread |
| 5 | Review "Scheduling job" bucket | Any contractors not responded >2 days? Send follow-up |
| 6 | Review "Awaiting invoice" bucket | Any completed jobs with no invoice >2 weeks? Follow up contractor |
| 7 | Check Tapi Inbox | Review messages from owners, tenants, contractors in Tapi messaging |
| 8 | Check PM email for owner responses | Any owners replied to inspection reports or approval requests via email? Process accordingly |
| 9 | Check Invoice Dashboard | Any new invoices arrived? Process matched/unmatched invoices |
| 10 | Cross-reference Inspection Express | Any finalised inspections today? Verify maintenance items synced to Tapi |
