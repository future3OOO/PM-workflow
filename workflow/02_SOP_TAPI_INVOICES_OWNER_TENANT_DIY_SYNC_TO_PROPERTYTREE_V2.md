# SOP — Tapi: Invoices (Owner vs Tenant vs DIY) + Sync to Property Tree (Version 2)

**Version:** V2 (Full)  
**Last updated:** 2026-02-24

---

## Steps
1) Confirm invoice matches the correct work order (work order # on invoice)  
2) Confirm job completion (notes/photos/tenant confirmation)  
3) Approve and code:
   - Owner charge (default)
   - Tenant charge (requires evidence + written comms trail)
   - DIY payment (owner pays directly)  
4) Notify owner invoice plan (paid from next rent funds / held if insufficient)  
5) Confirm invoice synced to Property Tree for Monday night payment run  
6) Close job with completion notes; file key artifacts to Property Tree if required
