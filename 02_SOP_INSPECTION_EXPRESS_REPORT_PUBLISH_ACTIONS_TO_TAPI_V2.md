# SOP — Inspection Express: Publish Report + Actions + Sync Maintenance to Tapi (Version 2)

**Version:** V2 (Full)

## Steps
1) Pre-publish quality check (photos clear, notes precise)  
2) Publish report and distribute to tenant and owner  
3) Create tenant action items with deadline (photos/by next inspection)  
4) Create maintenance items as needed and confirm they sync into Tapi  
5) File inspection report into Property Tree docs (04 Inspections)
